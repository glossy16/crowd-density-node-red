# Crowd Density Monitoring System (Node-RED)

## Overview
This project simulates an IoT-based crowd monitoring system using Node-RED.

## Features
- Density monitoring (Ultrasonic simulation)
- Motion detection (PIR simulation)
- Device count estimation (WiFi/Bluetooth simulation)
- Risk classification (LOW → CRITICAL)
- Live dashboard with charts, gauges, and alerts

## How to Run
1. Open Node-RED
2. Import `flows.json`
3. Click Deploy
4. Open http://localhost:1880/ui



## ⚠️ Note
This is a simulation-only project (no hardware used).
